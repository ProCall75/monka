# Réponse : "C'est de l'IA ?"

**Non.**

C'est un algo déterministe : une grille de notation automatique.

- **Déterministe** = Mêmes réponses → même résultat. Toujours.
- **Transparent** = Les règles sont lisibles dans le code.
- **Justifié** = Le barème vient d'échelles médicales validées (Zarit, HAS).

> Un médecin peut expliquer chaque score : "12/20 parce que réponse X à la question E1."

Pas de boîte noire. Pas de hasard. Pas de ChatGPT.

# Réponse : "Comment le modèle a été entraîné ?"

**Il n'a pas été entraîné. Ce n'est pas un modèle IA.**

C'est un système de règles écrites par des humains :

1. Des **experts métier** (IDEC, médecins) ont défini les critères de vulnérabilité
2. Ces critères ont été **traduits en règles** : "Si réponse X → +2 points"
3. Les règles sont **codées en dur** dans un fichier lisible

> C'est comme demander "comment le barème du bac a été entraîné ?" — Il n'a pas été entraîné, il a été **écrit** par des experts.

**Avantage** : Pas besoin de dataset, pas de biais statistique, 100% auditable.
