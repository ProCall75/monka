# APP/docs/pages — Fiches Audit Pages

Ce dossier contient une fiche par page de l'application Monka.

Chaque fiche documente :
- **Rôle** de la page
- **Données consommées** (hooks, getters, tables DB)
- **Composants utilisés**
- **Connexions DB vérifiées** (hardcode audit)
- **Métriques** (lignes, imports, composants)

Les fiches sont générées et mises à jour via le workflow `/iteration`.
