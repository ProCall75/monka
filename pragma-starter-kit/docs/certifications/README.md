## Comment ajouter une certification

1. Lancer le Quality Agent : `/quality-agent checkpoint=[checkpoint-name]`
2. Le rapport est sauvegardé dans ce dossier : `YYYY-MM-DD_checkpoint-name.md`
3. Mettre à jour ce tableau avec les résultats
